HTML

  • The Head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the body or structure of a webpage.

CSS

  • CSS adds decorative elements to the webpage.
  • it also structures how those elements are to be presented on the webpage

git

  • Git hub is a version control tool to allow users to pull code down make modifications and then push it back up to the main project.
  • Github uses pulls to sync up current versions of the code then allows the managers to see the edits and approve them for publishing to the main code.

javascript

  • Javascript is the part of the web design toolkit that allows for interactiveness on the webpage.
  • Javascript both adds elements such as links as well as things like widgets.
